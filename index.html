How to Download

Follow these steps:

  1. Click on the "Download BasicOS" button.
  2. It will redirect you to Google Drive where you can download the ISO file.
  3. Install Rufus from rufus.ie.
  4. Open Rufus.
  5. Select the downloaded ISO file.
  6. Do the conversion. WARNING: THIS WILL ERASE ALL DATA ON THE USB DRIVE
